# Custom Soft-Margin Support Vector Machine

The primary purpose behind building this model is to understand and demnostrate the underlying workings of SVM and its ability to find the best separating hyperplane and classify any given dataset. A custom Gradi- ent descent optimization algorithm was implemented to mimic the movement of a ball in a bowl. The model seemed to work reason- ably well and was 65 to 85% accurate in comparison with a prebuilt model from sklearn
